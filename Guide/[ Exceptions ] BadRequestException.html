# BadRequestException

  • an exception that will return a error message response.

Usage:

throw new \Pylon\Exceptions\BadRequestException("Opps, Bad Processing Request");
1
// output
{
	"error":"ERROR",
	"message":"Opps, Bad Processing Request"
}
1
2
3
4
5

# Parameters

Name Description
message Error Message that will be returned in response
status Http Status Code that will be returned in response (default: 400)

# Available Methods

# withErrorType()

  • Set the error type to be used for the response. ( default: ERROR )

# withValues()

  • Set the additional data to be used for the response. ( default: none )

# More Example

use Pylon\Exceptions\BadRequestException;
/* ...... */

	throw (new BadRequestException("Oops, Bad Processing Request",400))
		->withErrorType('BAD_PROCESSING_REQUEST')
		->withValues([
			"aaa"=>"Some value",
			"bbb"=>"Some value 2"
		]);
